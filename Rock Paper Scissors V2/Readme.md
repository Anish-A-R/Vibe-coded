# 🧠 AI System Overview

The AI uses:

* **Frequency Analysis**
* **Markov Chains (Order-1 & Order-2)**
* **Adaptive Aggression**
* **Prediction Countering**
* **Behavioral Learning**

The engine predicts your next move based on historical patterns and attempts to counter it intelligently.

Example prediction flow:

```text
Player History → Pattern Detection → Prediction → Counter Strategy
```

# 🔥 Firebase Setup

Replace the Firebase configuration in:

```javascript
const FB = {
  apiKey: "...",
  authDomain: "...",
  databaseURL: "...",
  ...
};
```

with your own Firebase project credentials.

Enable:

* Realtime Database
* Anonymous Authentication
